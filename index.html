html introduction

HTML is a standard markup languages for creating Web pages.

What is HTML? HTML stands for hyper text markup language , html is the standard markup language for creating web pages. HTML describes the structureof the webpage. HTML consists of a series of element tell the browser how to display the content.HTML elements label pieces of content HTML elements label pieces of content such as "things as a heading","This is paraghraph", "This is a link", etc

Example Explained The "!DOCTYPE HTML" deleration defines that this Document is an HTML5 Document

The "HTML" element is the root element of an HTTML page

The "head" element contains meta information about the HTML page

The "title" element specifies a title for a HTML page(which is shown in the browsers title bar or in the page's tab)

The "body" element defines the Document's body, and is a container for all the visible contents, such as headings paraghraphs, images, hyperlinks, table, lists, etc
